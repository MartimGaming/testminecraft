# Servidor Minecraft — gerido por CubePanel

Este repositório é controlado pelo painel CubePanel. **Não edites manualmente** os ficheiros em `.cubepanel/`.

## Estrutura
- `.devcontainer/` — config do GitHub Codespaces (Java 21 + Node)
- `.cubepanel/agent.js` — agente que liga o painel ao servidor
- `server/` — ficheiros do Minecraft (server.jar, world, plugins, etc.)
